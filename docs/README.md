# 기능 목록
- [x] 자동차 1대 전진 기능
- [x] 모든 자동차 전진 기능
- [x] 전진할 수 있는지 판단하는 기능(0-9 사이의 무작위 값이 4이상인 경우)
- [x] 무작위 값 선택하는 기능
- [x] 우승자 찾는 기능
- [x] 콤마(,) 기준으로 이름 분할하는 기능

## 입력
- [x] 경주 할 자동차 이름 입력하는 기능
- [x] 시도할 횟수 입력하는 기능

## 출력
- [ ] 각 차수별 실행 결과
- [ ] 우승자 안내 문구

## 예외
- [x] 자동차 이름이 4자를 초과하는 경우
- [ ] 시도할 횟수가 숫자가 아닌 경우
